Order Workflow Server Control

켜는법
- Linux/macOS: `python3 src/server.py`
- Windows: `py src/server.py`
- `PORT`를 지정하면 다른 포트로 실행할 수 있습니다.

끄는법
- 서버를 실행한 터미널에서 `Ctrl+C`를 누릅니다.
- systemd로 실행 중이면 `systemctl stop <service-name>`을 사용합니다.
- 종료할 때는 최신 스냅샷 백업 1개를 `data/backups/shutdown-latest/`에 만들고, 이전 종료 백업은 삭제합니다.
- 이 종료 스냅샷에는 존재하는 경우 `orders.json`, `users.json`, `audit.jsonl`이 포함됩니다.

참고
- 주문 데이터가 저장될 때 만드는 일별 백업은 그대로 유지됩니다.
